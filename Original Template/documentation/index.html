Installing Template

After unzip the download pack, you'll found a Template Folder ( Bryson ) with all the files. You can view this Template in any browser, you can display or edit the Template without an internet connection.

Now open your FTP Client (like Filezilla) and upload the content of the Template on your server root. Once the files are done uploading go to www.yourdomainname.com/index.html

To update count you need to care about 3 thing.
  1. data-from="0" // where from count will start
  2. data-to="12360" // Where count will end
  3. data-speed="2500" // how fast it will count
